# Open PDF in Zathura

Un plugin para Obsidian que abre el PDF activo en Zathura.

## Características

- 📄 Abre el PDF activo en Zathura.
- 📌 Recupera automáticamente la última página leída (gracias a Zathura).
- 🎯 Comando disponible desde la Paleta de comandos.
- 📎 Botón en la Ribbon de Obsidian.

## Requisitos

- Obsidian
- Zathura instalado en el sistema

## Desarrollo

Instalar dependencias:

```bash
npm install
```

Compilar y desplegar:

```bash
npm run deploy
```
